# Contributing to n8n Workflow Collection

Thank you for your interest in contributing! Here's how you can help.

## How to Contribute

### Adding a New Workflow

1. Fork this repository
2. Create a new folder under `workflows/` with a descriptive name
3. Include the following files:
   - `workflow.json` — the n8n workflow export
   - `README.md` — setup instructions, required credentials, and use case description
4. Submit a Pull Request with a clear description

### Reporting Issues

- Use GitHub Issues to report broken workflows or suggest improvements
- Include the workflow name and the error you encountered

### Suggesting Workflows

- Open an Issue with the tag `[REQUEST]` and describe the automation you'd like to see

## Guidelines

- Test your workflow before submitting
- Remove any personal API keys or credentials from the JSON
- Write clear documentation for setup steps
- Use descriptive folder and file names

## Code of Conduct

Be respectful, constructive, and helpful. We're all here to learn and automate.
